EDTC ship 3D models — drop files here
========================================

Put one model per ship in this folder named EXACTLY <id>.glb
(.gltf and .obj also work). It is auto-detected and bundled on the
next build. Rendered as an orange WIREFRAME, so low-poly / untextured
is totally fine. Prefer uncompressed glTF (no Draco).

Any ship without a file here falls back to the 2D schematic.
